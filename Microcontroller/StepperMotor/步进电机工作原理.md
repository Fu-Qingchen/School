# 步进电机工作原理

根据课本，单双八拍方式为
$$
A-AB-B-BC-C-CD-D-DA
$$

| 通电 |  1B  |  2B  |  3B  |  4B  | 代码 |
| :--: | :--: | :--: | :--: | :--: | :--: |
|  A   |  1   |  0   |  0   |  0   | 0x01 |
|  AB  |  1   |  1   |  0   |  0   | 0x03 |
|  B   |  0   |  1   |  0   |  0   | 0x02 |
|  BC  |  0   |  1   |  1   |  0   | 0x06 |
|  C   |  0   |  0   |  1   |  0   | 0x04 |
|  CD  |  0   |  0   |  1   |  1   | 0x0C |
|  D   |  0   |  0   |  0   |  1   | 0x08 |
|  DA  |  1   |  0   |  0   |  1   | 0x09 |
